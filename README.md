# 🎮 Lobisomem: A caçada

> Um jogo 2D estilo plataforma inspirado em clássicos como Mario, com sistema de combate, inimigos e progressão.

---

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um jogo 2D onde o jogador controla um personagem que deve atravessar o cenário, enfrentar inimigos e alcançar o objetivo final.

O jogo foi desenvolvido com foco em:
- Física básica (gravidade e pulo)
- Sistema de combate
- Inteligência simples de inimigos
- Estrutura modular de código

---

## 🚀 Funcionalidades

- 🎮 Movimentação do jogador (andar e pular)
- 👾 Sistema de inimigos com diferentes comportamentos
- 💥 Sistema de combate e dano
- 🩸 Efeitos visuais (ex: sangue ao receber dano)
- ❤️ Sistema de vidas (HUD)
- 🎵 Sons e música de fundo
- 🏁 Sistema de vitória e derrota

---

## 🕹️ Como Jogar

- Use as teclas para movimentação
- Pule obstáculos
- Evite ou elimine inimigos
- Chegue até o final da fase para vencer

---

## 🧱 Estrutura do Projeto

- main.py # Loop principal do jogo
- game_logic.py # Lógica do jogo
- audio.py # Sistema de áudio
- tela_vitoria_derrota.py # Telas finais
- assets/ # Imagens e recursos
- tests/ # Testes automatizados

---

## ⚙️ Tecnologias Utilizadas

- Python
- OpenGL
- GLFW
- PIL / NumPy
